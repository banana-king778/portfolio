Loan Bot

Stored and managed loans. Had member "ranks" that decided the interest rates and other terms of the loans. The bot reminded borrowers to make payments and also reminded them how much they owe. The bot was scrapped because the server became inactive. The bot was written with Python using the Discord.py library.

TimeBot

Accepted time requests and converted them into Coordinated Universal Time, so the discord server was able to effectively schedule and coordinate events and activities. This bot was very basic and when the server became inactive I again scrapped it. The bot was written with Python using the Discord.py library.

SuggestionBot

Accepted suggestions for songs and prompted "DJs" to accept or deny the suggestion. Based on that acceptance or denial the bot would then play the music in a "radio" channel or deny it. This bot was also scrapped as the server became inactive. The bot was written with Python using the Discord.py and youtube-dl library.

BananaBot

Personal project that automates form submissions and Unix timestamps within discord. Uses modals to accept form submissions that are sent to google forms, and convert time values into Unix timestamps to be used within discord. The bot was written with Python using the Discord.py(v2) library, and other miscellaneous libraries.
